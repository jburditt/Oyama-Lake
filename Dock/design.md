# DOCKS

## Dock A - Ramp to land
- Plank - 2" thickness, 101" length, any width 
- Fascia - 7 3/4" width, 210 3/4" (length touching land is 211") length, 2" thickness
- Centre Beam - 2" thickness, 5" width 
- Side beams - 4" x 4"
- End pieces - 101 1/2" length, 2" thickness 

## Dock B - T shape
- End pieces - 105" length, 2" thickness
- Fascia (sides) - 209 1/2" length
- Beams - 4" width, 6" height
- Braces (6) between barrels and end pieces - 4" width, 2" thickness
